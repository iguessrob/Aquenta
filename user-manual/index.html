Purpose, Audience, and Basic Use

Purpose and Scope

This manual explains how to complete common tasks in the Aquenta Water Billing System. It focuses on user actions, not software development.

  • Covers: login, customer records, billing, payments, and reports.
  • Does not cover: database scripts, server setup, and source code edits.

Clear Terms

Concessioner
The account holder connected to a water meter and billing record.
Arrears
Past due balance from previous periods.
Tariff
The approved rate used to calculate charges.
Consistency rule: This manual always uses the same action words: Select, Type, and Save.

Tasks: Sign In and Add Customer

Task 1: Sign In Safely

  1. Open the Aquenta login page.
  2. Type your username in the Username field.
  3. Type your password in the Password field.
  4. Select Sign In.
  5. Confirm that the dashboard opens and your name appears in the header.

Expected result: Dashboard modules are available.

Task 2: Add a Customer Record

  1. Select Customer Records from the dashboard.
  2. Select Add Concessioner.
  3. Type complete customer details (name, address, meter information).
  4. Review entries for spelling and number accuracy.
  5. Select Save.

Expected result: The new customer appears in the records list.

Tasks: Generate Bill and Record Payment

Task 3: Generate a Bill

  1. Select Billing from the menu.
  2. Search and select the customer account.
  3. Type latest meter reading and billing period.
  4. Verify tariff and prior balance.
  5. Select Generate Bill.
  6. Review invoice summary before finalizing.
Warning: Do not generate duplicate bills for the same period.

Task 4: Record a Payment

  1. Select Payment.
  2. Find the customer using account number or name.
  3. Select the correct invoice.
  4. Type payment amount, date, and reference.
  5. Select Save.
  6. Print or download receipt if required.

Expected result: Balance and payment history are updated.

Task: Reports and Screenshot Snippets

Task 5: Run a Report

  1. Select Reports.
  2. Choose the report type (example: Total Consumption, Arrear Summary).
  3. Set date range and filters.
  4. Select Generate.
  5. Review totals and export if needed.

Expected result: The selected report displays correctly.

Troubleshooting, FAQ, and Consistency Rules

Common issues and direct solutions
Issue Possible Cause Action
Cannot sign in Wrong credentials or expired password Type credentials again carefully. If it fails, request password reset.
Customer not visible in billing Record not saved or active filter is hiding it Confirm customer exists in records and clear filters.
Bill total is incorrect Wrong reading, tariff, or billing period Check values then generate the bill again.
Payment did not reduce balance Payment posted to wrong invoice Open payment history, verify invoice ID, and correct entry.

Consistency Checklist

  • Use exact menu names as shown on screen.
  • Follow task steps in order.
  • Document page name, account ID, and error message when asking support.
